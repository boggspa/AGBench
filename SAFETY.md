# Safety Guidelines

This application wraps an AI agent capable of modifying local files and executing shell commands.

## Implemented Guardrails
- **Workspace Confinement**: Operations are only allowed within the explicitly selected workspace directory. Path traversal (`../`) is blocked in diffs and operations where possible.
- **Denylist**: The following files and paths are blocked from access by the UI layer (and we warn the user if the workspace contains them):
  - `.env`, `.env.*`
  - `.git` credentials
  - `~/.ssh`
- **Visibility**: Subprocess stdout/stderr must be visible. Diffs are shown before/after tasks for manual review.

## Manual Review
Always review the `git diff` generated by the agent before committing changes. Dangerous commands are not yet intercepted at the Electron layer, but the Gemini CLI handles its own approval flows (unless YOLO mode is on).
